Introduction

The RStudio IDE has many features and options to help you work in the way that best suits your needs.

Many of the features discussed in this article require RStudio 1.3 or newer.

Email accessibility-related questions and comments to accessibility@rstudio.com.

Zoom Level

The size of text and other RStudio user-interface elements can be adjusted using the Zoom feature.

For RStudio Desktop, use the "Zoom In", "Zoom Out", or "Actual Size" commands on the View menu.

RStudio Server is zoomed using your web browser's page zoom feature. Consult browser help for more details.
